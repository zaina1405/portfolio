# Security Policy

## Supported Versions

This is a personal portfolio website, not a versioned software project — there's only ever one live version, kept up to date on the `main` branch.

## Reporting a Vulnerability

If you find a security issue with this site (for example, an exposed file, a broken link that could be exploited, or anything else concerning), please reach out directly:

- Email: zain.salman@hotmail.com

I'll look into it as soon as I can and let you know the outcome. Since this is a static site with no backend, database, or user accounts, the scope for vulnerabilities is limited — but I appreciate any reports.
